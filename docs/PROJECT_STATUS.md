# Project Status

## Completed in this public edition

- typed FastAPI endpoints;
- request validation;
- learner-profile compatibility rules;
- deterministic planning and generation;
- answer-key creation;
- output validation;
- responsive browser interface;
- automated API and rule tests;
- continuous integration workflow.

## In progress in the private product

- stable Gemini-provider integration;
- richer worksheet rendering;
- image and asset handling;
- persistence, users, and account systems;
- contribution library and review workflow.

## Intentionally excluded

- API keys and secrets;
- proprietary prompts;
- private business and pricing documents;
- private test data;
- production deployment configuration;
- unfinished features presented as complete.

## Engineering approach

The project is developed incrementally with a QA mindset: explicit constraints, reproducible demo behaviour, validation before display, test fixtures, and honest separation between working functionality and future work.
